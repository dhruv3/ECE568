To run this program in Eclipse you need to set Arguments for ServerSide.java and ClientSide.java.
You can do that as follows:
# For ServerSide.java:
1. In the menu click "Run" > Choose "Run Configurations".
2. Find "ServerSide.java" application using search filter
3. Select Arguments tab, and type “5500”
4. Click Run.

# For Client.java:
1. In the menu click "Run" > Choose "Run Configurations".
2. Find "ClientSide.java" application using search filter
3. Select Arguments tab, and type “localhost” and “5500” (Note both these arguments should be separated by a single space)
4. Click Run.
